## 2. UI/UX Design Tokens

### 2.1 Color Palette
| Token | Hex Code | Usage |
| :--- | :--- | :--- |
| `color-sky-blue` | `#0033A0` | Primary Brand Color (Soyombo Blue), Accents |
| `color-gold` | `#FFD700` | Secondary Accents, Icons, Call-to-Action |
| `color-steppe-green` | `#4F6348` | Background sections, subtle borders |
| `color-earth-tan` | `#D2B48C` | Secondary backgrounds, cards |
| `color-deep-slate` | `#1A1A1A` | Primary Typography |
| `color-off-white` | `#F9F9F9` | Main Page Background |

### 2.2 Typography
*   **Heading Font:** *Playfair Display* (Serif) - To evoke a sense of history and majesty.
*   **Body Font:** *Inter* (Sans-Serif) - For high readability and modern feel.
*   **Scale:**
    *   `h1`: 4rem / 700 weight (Hero)
    *   `h2`: 2.5rem / 600 weight (Section Headers)
    *   `body`: 1.125rem / 400 weight (Paragraphs)

### 2.3 Spacing & Elevation
*   **Base Unit:** `8px`
*   **Container Max-Width:** `1200px`
*   **Section Padding:** `80px` (Vertical) to maintain "vast" feel.
*   **Shadows:** Soft `0 4px 20px rgba(0,0,0,0.05)` for cards.

## 3. Component Breakdown

### 3.1 `Navigation` (Global)
*   **Type:** Sticky, transparent background that turns `color-off-white` on scroll.
*   **Elements:** Logo (Soyombo symbol), Links (History, Landscapes, Culture), "Visit" CTA button.

### 3.2 `HeroSection`
*   **Visual:** Full-screen high-resolution background video or image of the Mongolian Steppe.
*   **Content:** 
    *   H1: "The Last Great Wilderness."
    *   Subtext: "Experience the nomadic spirit of the eternal blue sky."
    *   Primary CTA: "Begin the Journey."

### 3.3 `QuickStats` (Grid)
*   **Layout:** 4-column horizontal bar.
*   **Data Points:** 
    *   Population Density (Lowest in the world).
    *   Sunshine Days (250+ days/year).
    *   Horse-to-Human Ratio.
    *   Capital City (Ulaanbaatar).

### 3.4 `RegionCards` (Interactive)
*   **Structure:** Horizontal scroll or 3-column grid.
*   **Components:** 
    *   **Gobi Desert:** Focus on "Flaming Cliffs."
    *   **Khuvsgul Lake:** Focus on "The Blue Pearl."
    *   **Altai Mountains:** Focus on "Eagle Hunters."
*   **Interaction:** Hover effect expands card to show brief 2-sentence description.

### 3.5 `CultureFeature` (Split Layout)
*   **Left:** High-quality image of a traditional *Ger* (Yurt).
*   **Right:** Text content detailing the "Three Manly Games" of Naadam and nomadic hospitality.

### 3.6 `Footer`
*   **Content:** Simple copyright, social icons, and a "Learn More" link to official Mongolian Tourism boards.

## 4. Technical Architecture

### 4.1 Tech Stack
*   **Framework:** Next.js (for SEO and image optimization).
*   **Styling:** Tailwind CSS (using the design tokens above).
*   **Animations:** Framer Motion (Subtle "fade-in-up" transitions as the user scrolls to mimic the horizon appearing).
*   **Icons:** Lucide React (Minimalist line icons).

### 4.2 Performance Requirements
*   **Image Compression:** All hero assets must be WebP format.
*   **Lighthouse Score Target:** 95+ for Performance and Accessibility.
*   **Responsive Breakpoints:**
    *   Mobile: `< 640px` (Single column, hamburger menu).
    *   Tablet: `640px - 1024px` (Two column grid).
    *   Desktop: `> 1024px` (Full design layout).
